# Yume
## DirectX 12 / Vulkan Graphics Engine
The engine currently supports following features:
- DirectX 12 Framework
- Logging System
- Event System
- Timer System
- Mouse Input System
- Premake Meta Build System

## Build instructions
(**Note**: Only supports Windows)
- DirectX 12
  - Clone the repo
  - Run `setup.bat` file
- Vulkan
  - Follow [(feature/vulkan)](https://github.com/pratik-dhende/Yume/tree/feature/vulkan) README.md
