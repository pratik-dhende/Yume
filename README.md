# Yume
## DirectX 12 Graphics Engine
The engine is currently supports following features:
- DirectX 12 Framework
- Logging System
- Event System
- Timer System
- Mouse Input System
- Premake Meta Build System

## Build instructions
(**Note**: Only supports Windows)
- Clone the repo
- Run `setup.bat` file
