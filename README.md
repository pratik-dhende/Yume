## Features
- Vulkan framework
- Frames In Flight
- Resource Manager
- Service Locator
- CMake Meta Build System

## Requirements
- Vulkan SDK (>= 1.4.x)
- CMake
- vcpkg

## Build instructions
(**Note**: Only supports Windows)
- Clone the repo
- Switch branch to `feature/vulkan`
- Configure CMake, build and run
